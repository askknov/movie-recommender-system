# Movie Recommender System

This project builds a **Movie Recommendation System** using the TMDB 5000 Movies Dataset.  
It recommends movies similar to a given movie using **content-based filtering**.

## Dataset
The dataset used:
- tmdb_5000_movies.csv
- tmdb_5000_credits.csv

Source: TMDB 5000 Movie Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## How the System Works
1. Data preprocessing and cleaning
2. Feature extraction from movie metadata
3. Creating movie vectors
4. Calculating similarity using cosine similarity
5. Recommending similar movies

## Example
If a user selects a movie like **Avatar**, the system recommends similar movies based on features such as genre, keywords, cast, and crew.
